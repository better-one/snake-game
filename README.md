# 贪吃蛇游戏

🐍 经典的贪吃蛇游戏，使用 Python + Pygame 开发。

## 🎮 游戏截图

支持 6 种精美皮肤，每种都有独特的视觉风格！

## ✨ 功能特性

- [x] 基本贪吃蛇移动
- [x] 食物生成（含金色食物）
- [x] 碰撞检测
- [x] 计分系统
- [x] 难度递增 ⭐
- [x] 皮肤系统 ⭐
- [ ] 排行榜
- [ ] 多人模式

## 🚀 快速开始

```bash
# 安装依赖
pip install pygame

# 运行游戏
python snake.py
```

## 🎹 控制方式

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → 或 W A S D | 控制移动方向 |
| 空格 | 开始/暂停游戏 |
| M | 更换皮肤 |
| ESC | 退出游戏 |

## 🎨 可用皮肤

| 皮肤 | 名称 | 特点 |
|------|------|------|
| classic | 经典绿 | 传统贪吃蛇配色 |
| blue | 冰蓝 | 清爽蓝色主题 |
| purple | 紫魅 | 神秘紫色风格 |
| orange | 橙热 | 活力橙色调 |
| matrix | 黑客帝国 | 绿色代码风 |
| dark | 暗黑 | 低调暗色系 |

**按 M 键即可切换皮肤！**

## 📈 游戏机制

- **计分**：普通食物 +10 分，金色食物 +20 分
- **升级**：每 50 分升一级
- **加速**：每级速度 +2 FPS，最高 25 FPS
- **金色食物**：10% 概率出现
