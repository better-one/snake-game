# 贪吃蛇游戏

🐍 经典的贪吃蛇游戏，使用 Python + Pygame 开发。

## 版本历史

- v1.1.0 - 难度递增：速度随分数提升，新增金色食物
- v1.0.0 - 初始版本：基础贪吃蛇功能

## 功能特性

- [x] 基本贪吃蛇移动
- [x] 食物生成
- [x] 碰撞检测
- [x] 计分系统
- [x] 难度递增 ⭐ NEW
- [ ] 皮肤系统
- [ ] 排行榜

## 快速开始

```bash
# 安装依赖
pip install pygame

# 运行游戏
python snake.py
```

## 控制方式

- ↑ ↓ ← → 或 W A S D：控制方向
- 空格键：开始/暂停
- ESC：退出游戏
